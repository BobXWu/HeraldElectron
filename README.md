#小猴偷米PC端版
<br>
##说明
1. 此项目旨在用PC端上实现小猴偷米的各种功能，拓展小猴偷米的使用覆盖范围
2. 技术方面,此项目用html,css,javascript实现，再用Electron框架打包成windows平台，MacOS平台上客户端程序。后台方面直接使用heraldstudio的接口

##需求分析
1. 登录
	>用户填写一卡通号和密码，并提供保持登录状态的功能
2. 注销
	>注销，并清除
2. 查询srtp学分
3. 查询跑操信息
4. 查询自己课表
5. 查询人文讲座信息
6. 查询人文讲座预告
7. 查询一卡通信息
8. 查询seu-wlan信息
9. 查询图书馆借书情况
10. 图书馆续借功能
10. 查询考试安排信息
11. 查询教务信息
12. 查询体侧信息
13. 连接和断开seu-wlan
